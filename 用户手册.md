# 用户手册

项目名称：智投社区：股票基金投资论坛系统

本文档面向课程演示用户，说明系统主要页面、账号角色和核心操作流程。

## 一、访问方式

本地部署完成后，在浏览器访问：

```text
http://127.0.0.1:5173
```

后端接口文档访问：

```text
http://127.0.0.1:8000/docs
```

## 二、演示账号

演示账号密码均为 `Admin123456`。

| 角色 | 邮箱 | 说明 |
| --- | --- | --- |

## 三、首页浏览

进入系统后，首页展示股票基金投资论坛的主要内容：

| 区域 | 功能 |
| --- | --- |
| 顶部导航 | 首页、板块、热榜、群组、关注、通知、个人中心等入口 |
| 板块入口 | 查看 A 股、港股、美股、基金、量化、价值投资等讨论区 |
| 最新帖子 | 浏览近期讨论内容 |
| 热门讨论 | 查看浏览、点赞、评论较多的帖子 |

未登录用户可以浏览首页、板块、帖子详情、搜索结果和热榜。

## 四、注册与登录

### 1. 登录

1. 点击顶部导航中的登录入口。
2. 输入演示邮箱和密码。
3. 登录成功后返回系统页面。

### 2. 注册

1. 进入注册页面。
2. 选择手机号或邮箱注册。
3. 输入验证码、密码等信息。
4. 提交后完成注册。

课程演示环境中的验证码为模拟流程，不接入真实短信或邮件服务。

## 五、个人中心

登录后进入个人中心，可以完成以下操作：

| 功能 | 说明 |
| --- | --- |
| 查看资料 | 查看昵称、简介、认证等级、风险偏好、积分和等级 |
| 编辑资料 | 修改昵称、头像、简介、投资经验、关注市场和隐私设置 |
| 实名认证 | 提交实名信息，演示环境中为模拟通过 |
| 专业认证 | 提交专业资质，专业用户可用于长文分析 |
| 风险测评 | 根据问卷结果生成风险偏好 |
| 我的收藏 | 查看已收藏帖子 |
| 通知中心 | 查看评论、回复、关注、提及等站内通知 |
| 私信 | 查看和发送站内私信 |

## 六、论坛内容操作

### 1. 查看板块

进入“板块”页面后，可以查看全部讨论区。点击某个板块可进入板块详情页，查看该板块下的帖子列表。

### 2. 发布帖子

1. 登录普通用户或专业用户账号。
2. 进入发帖页面。
3. 选择板块，填写标题、正文和标签。
4. 可按页面入口补充投票选项。
5. 提交后进入帖子详情或列表刷新。

### 3. 长文分析

长文分析面向专业认证用户。普通用户提交长文分析时会被权限限制，专业用户可以发布分析类内容。

### 4. 搜索

在搜索入口输入关键词，可以检索帖子标题和正文。搜索结果页会展示相关帖子，并提供搜索建议。

### 5. 热榜

热榜页面展示热门帖子和每日、每周热议话题。热度主要根据浏览、点赞和评论等数据综合排序。

## 七、互动功能

登录后可在帖子详情页执行以下操作：

| 功能 | 说明 |
| --- | --- |
| 评论 | 对帖子发表回复 |
| 楼中楼回复 | 对某条评论继续回复 |
| 点赞 | 对帖子或评论点赞，再次点击可取消 |
| 收藏 | 收藏帖子，再次点击可取消 |
| 关注作者 | 关注帖子作者 |
| 特别关注 | 将已关注用户标记为特别关注 |
| 举报 | 举报帖子或评论，举报会进入后台处理队列 |
| 投票 | 对投票帖的选项进行投票 |
| 附件查看 | 查看帖子关联附件信息 |

评论内容中出现 `@昵称` 时，系统会生成提及通知。

## 八、群组功能

群组页面支持以下流程：

| 功能 | 说明 |
| --- | --- |
| 群组列表 | 查看公开群组 |
| 群组详情 | 查看群组简介、成员数、讨论和资源 |
| 创建群组 | 登录用户可创建公开、审核或私密类型群组 |
| 加入群组 | 公开群可直接加入，审核群会生成申请状态 |
| 退出群组 | 已加入用户可退出群组 |
| 群组讨论 | 在群组内发布讨论内容 |
| 群组资源 | 查看或新增群组资料信息 |

## 九、后台管理

管理员账号登录后，顶部导航显示后台入口。普通用户不会显示后台入口，直接访问后台路由也会被拦截。

后台功能包括：

| 功能 | 说明 |
| --- | --- |
| 后台概览 | 查看用户、帖子、举报、审核等统计摘要 |
| 内容审核 | 查看待审核内容并进行通过或驳回 |
| 举报处理 | 处理用户对帖子或评论的举报 |
| 敏感词管理 | 查看和启停敏感词规则 |
| 用户处置 | 查看用户处罚记录 |
| 运营统计 | 查看热门话题和活跃板块 |

## 十、退出登录

在页面中执行退出操作后，前端会清理本地登录状态。再次访问个人中心、发帖、通知、私信和后台等页面时，需要重新登录。

## 十一、演示建议流程

课程验收时建议按以下顺序演示：

1. 打开首页，浏览板块、帖子和热榜。
2. 使用普通用户登录，发布帖子并完成评论、点赞、收藏、关注和私信。
3. 进入群组页面，演示加入群组、群组讨论和群组资源。
4. 使用专业用户登录，演示长文分析。
5. 使用管理员登录，进入后台查看审核、举报、敏感词和统计。
6. 打开 `http://127.0.0.1:8000/docs` 展示后端接口文档。

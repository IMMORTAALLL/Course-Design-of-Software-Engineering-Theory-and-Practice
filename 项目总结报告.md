# 项目总结报告

项目名称：智投社区：股票基金投资论坛系统

## 一、项目概述

智投社区是一个面向股票和基金投资者的论坛系统，支持用户注册登录、投资内容发布、评论互动、关注收藏、群组交流、私信通知以及后台审核管理。系统采用前后端分离架构，后端使用 FastAPI + SQLAlchemy，前端使用 Vue 3 + Vite + TypeScript，开发和验收环境默认使用本地 SQLite。

## 四、技术架构总结

后端采用 FastAPI 提供 RESTful API，通过 SQLAlchemy 管理数据库模型和查询逻辑，通过 JWT Bearer Token 完成登录态校验。业务代码按模块拆分为认证、论坛、互动和后台四个目录，公共响应、异常处理和依赖注入放在 `backend/app/common`。

前端采用 Vue 3 + Vite + TypeScript，按业务模块划分页面、API 和类型定义。路由层区分公开页面、登录后页面和管理员页面，接口层统一处理后端 `code/message/data` 响应格式。

数据库默认采用 SQLite，便于课程验收本地冷启动；同时保留 MySQL 8.0 建表脚本和种子数据，便于后续部署扩展。
